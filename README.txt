Project: Website Profil Perusahaan Sederhana
Deskripsi:
Website ini dibuat menggunakan HTML, CSS, dan JavaScript tanpa framework.
Tujuan: Menampilkan profil perusahaan, struktur organisasi, produk, dan kegiatan sosial.

Struktur Folder:
- index.html → Halaman utama
- style.css → Desain/tampilan
- script.js → Logika navigasi
- img/ → Folder gambar (logo)

Cara Menjalankan:
1. Buka folder 'Website-Perusahaan' di VS Code.
2. Klik kanan pada 'index.html' → pilih 'Open with Live Server' (jika ada ekstensi Live Server).
   Atau buka manual di browser.
3. Edit isi file sesuai data perusahaanmu.
